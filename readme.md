## Getting Started

2. Clone the repo
   ```sh
   git clone https://github.com/Izzycl/enara-tt
   ```
3. Install YARN packages
   ```sh
   yarn install
   ```
4. Run
   - iOS
   ```sh
   yarn ios
   ```
   - Android
   ```sh
   yarn android
   ```
